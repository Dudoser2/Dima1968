## Описание проекта
Приложение «Training Planner» — это планировщик тренировок с графическим интерфейсом. Программа позволяет сохранять информацию о тренировках, включая дату, тип тренировки и длительность в минутах. Приложение автоматически проверяет корректность ввода, сохраняет базу данных в файл `trainings.json` и поддерживает фильтрацию записей.

## Инструкция по запуску

### Требования
* Python 3.x или выше.

### Запуск
1. Убедитесь, что у вас установлен Python.
2. Скачайте файл `main.py` в отдельную папку.
3. Откройте терминал/консоль в этой папке и запустите команду:
```bash
python main.py
